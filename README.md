Elixir client for [Crux DB](https://www.opencrux.com), the general purpose
database with bitemporal SQL and Datalog.

## Installation

The package can be installed by adding `translixir` to your list of
dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:translixir, "~> 0.1.0"}
  ]
end
```
